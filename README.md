# MMT-Projekt
Eigenständig entwickeltes Elektronikprojekt im Rahmen des Physik-Bachelorstudiums an der Universität Potsdam. 
Es ist Teil des Moduls Moderne Messtechnik und hat das Ziel, physikalische Sensoren elektronisch auszulesen und deren Daten zu analysieren.
